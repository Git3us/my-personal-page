# UI Guidelines (Figma -> Vue)

## Design tokens
- Primary: `#2563EB`
- Accent: `#38BDF8`
- Dark background: `#0F172A`
- Light background end: `#F8FAFC`

## Motion
- Hover duration: `180ms ~ 240ms`
- Card lift: `translateY(-3px)`
- Theme transition: `0.4s`

## Components in current app
- `Hero`
- `SectionCard`
- `Chip` / `Chip outline`
- `Theme toggle button`
- `ItemCard` (projects/blogs/contacts)

## File convention
- Tokens & theme: `frontend/src/styles/theme.css`
- Typography: `frontend/src/styles/fonts.css`
- Layout reset/base: `frontend/src/styles/base.css`
- Component visuals/motion: `frontend/src/styles/components.css`
